# This is a simple Ai called James

A basic AI assistant built with Python that uses text-to-speech functionality to interact with users. The project demonstrates how to create a simple voice-enabled assistant capable of responding to commands and speaking output using the `pyttsx3` library.

## Features

* Text-to-speech responses
* Voice interaction with users
* Lightweight and beginner-friendly
* Runs locally without requiring internet access

## Technologies Used

* Python
* pyttsx3

## Skills Demonstrated

* Python Programming
* Automation
* Text-to-Speech (TTS)
* Voice Assistant Development
* Problem Solving

## Project Goal

The goal of this project was to gain hands-on experience with Python automation and voice-enabled applications by building a simple AI assistant from scratch.
